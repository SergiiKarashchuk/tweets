## Description
Welcome, to application Tweets. It loads user cards
from the database. By clicking on the Follow button in the card, you can
subscribe to a user or unsubscribe.
## Technologies
- **react**
- react-router-dom
- styled-emotions
- axios
- local-storage
- mockapi
